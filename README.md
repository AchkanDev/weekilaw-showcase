## 📱 App Overview

**WeekiLaw** is a professional cross-platform mobile application built with Flutter that provides users with smart legal consultations, legal documents, and expert guidance — all in one place.

## ✨ Key Features

- ⚖️ **Smart Legal Consultation** — Connect with certified lawyers
- 📄 **Legal Document Library** — Access hundreds of legal templates
- 🔔 **Smart Notifications** — Stay up to date on your cases
- 👤 **User Profiles** — Manage your personal and legal information
- 🔐 **Secure Authentication** — Firebase Auth (Email, Google, Apple)
- 🌐 **Cross-Platform** — Android & iOS from a single codebase
- 🎨 **Pixel-Perfect UI** — 28+ beautifully designed screens

## 🏗️ Architecture & Tech Stack

```
lib/
├── core/                    # Core utilities, theme, constants
├── features/
│   ├── auth/                # Authentication (BLoC + Firebase)
│   ├── consultation/        # Lawyer consultation flow
│   ├── documents/           # Legal document management
│   └── profile/             # User profile management
└── shared/                  # Shared widgets & components
```

| Layer | Technology |
|-------|-----------|
| **UI** | Flutter + Custom Design System |
| **State** | BLoC / Cubit |
| **Architecture** | Clean Architecture |
| **Backend** | Firebase (Firestore, Auth, Storage) |
| **Navigation** | Go Router |
| **DI** | GetIt + Injectable |
