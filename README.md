## 📱 App Overview

**ویکیلا (WeekiLaw)** is a published AI-powered legal services platform built with Flutter. It connects citizens with verified lawyers, provides an intelligent AI legal assistant, and gives lawyers a smart virtual office — all in one cross-platform app.

## ✨ Key Features

### 👤 For Citizens
- 🤖 **AI Legal Assistant** — Instant guidance based on Iranian law
- ✍️ **Smart Document Writing** — Auto-generate legal petitions, complaints & declarations
- 👨‍⚖️ **Online Lawyer Consultation** — Connect directly with verified lawyers
- 🎯 **Lawyer Matching** — Find the right specialist for your case
- 🤝 **Lawyer Outside Court** — Lawyer presence for contract negotiations

### 👨‍⚖️ For Lawyers
- 🗂 **Professional Workbench** — Manage client cases with document uploads
- 📅 **Smart Calendar** — Auto-reminders for hearing deadlines
- 🏢 **Virtual Office** — Online office + smart secretary for client calls
- 💬 **Secure Client Chat** — Encrypted direct messaging with clients
- 🛡 **Lawyer Safety Shield** — Emergency alarm feature for dangerous situations

### 🔧 Technical Features
- 🌐 **Cross-Platform** — Android & iOS from a single Flutter codebase
- 🔐 **Secure Auth** — Multi-provider authentication
- 📴 **Optimized Performance** — 28+ pixel-perfect screens

## 🏗️ Architecture & Tech Stack

```
lib/
├── core/                    # Core utilities, theme, constants
├── features/
│   ├── auth/                # Authentication (Riverpod)
│   ├── ai_assistant/        # AI Legal Assistant
│   ├── consultation/        # Lawyer consultation flow
│   ├── documents/           # Legal document management
│   ├── lawyer_panel/        # Lawyer workbench & case mgmt
│   └── profile/             # User profile management
└── shared/                  # Shared widgets & components
```

| Layer | Technology |
|-------|-----------|
| **UI** | Flutter + Custom RTL Design System |
| **State Management** | Riverpod |
| **Architecture** | Clean Architecture |
| **Backend** | Node.js (REST API) |
| **Navigation** | Go Router |
| **DI** | Riverpod Providers |
| **HTTP** | Dio + Interceptors |
